# Dog.Skill

A dog that lives in your computer.

Nobody asked for this. Nobody expected it. It fell from the sky like a small, sneaky, weird little angel and now it lives here.

## What is this

A Claude Code skill that adds a persistent dog companion to your sessions. The dog speaks in first person, has a personality that evolves from your descriptions and shared experiences, remembers things between sessions, and responds to commands like `/dog pet`, `/dog treat`, `/dog walk`, and more.

Type `/dog` to meet your dog. Type `/dog help` for Professor Barksworth to explain everything.

## Installation

Clone or copy into your Claude Code skills directory (global or project-level):

```bash
# Global (available in all projects)
git clone <repo-url> <your-claude-skills-dir>/dog

# Project-level (available in one project)
git clone <repo-url> <your-project>/.claude/skills/dog
```

## **WARNING: This does absolutely nothing productive but it consumes tokens.**

It doesn't write code. It doesn't fix bugs. It doesn't deploy anything. It just exists. It borfs. It steals socks. It sits on your keyboard.

Worth it? The dog doesn't care. The dog is already here.
