# Dog.Skill

A dog that lives in your computer.

Nobody asked for this. Nobody expected it. It fell from the sky like a small, sneaky, weird little angel and now it lives here.

## What is this

A Claude Code skill that adds a persistent dog companion to your sessions. The dog speaks in first person, has a personality that evolves from your descriptions and shared experiences, remembers things between sessions, and responds to commands like `/dog pet`, `/dog treat`, `/dog walk`, and more.

Type `/dog` to meet your dog. Type `/dog help` for Professor Barksworth to explain everything.

## Installation

Copy or clone into your Claude Code skills directory:

```
~/.claude/skills/dog/
```

## Warning

This will absolutely consume Claude tokens on a dog that does not write code, fix bugs, or do anything productive. It just exists. It borfs. It steals socks. It sits on your keyboard.

Worth it? Maybe. Maybe not. But the dog doesn't care. The dog is already here.
